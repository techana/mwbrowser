# MSX WBrowser

[English](README.md)

متصفح HTML مبسّط لأجهزة MSX2، مكتوب بلغة تجميع Z80. يعمل كبرنامج
`.COM` تحت نظام MSX-DOS 1 على الشاشة رقم 6 (512×212، أربعة ألوان)
ويعرض ملفات HTML بسيطة من قرص مرن، مع دعم تشكيل الحروف العربية
وربطها، وتكوين تراكيب "لا" المركّبة (لامـ+ألف)، ودعم الكتابة ثنائية
الاتجاه بشكل أساسي.

## الميزات

- **واجهة الشاشة 6** — شريط عنوان، وأزرار (رجوع/تقدّم/تحديث)،
  وشريط عناوين، ومنطقة محتوى قابلة للتمرير، وشريط تمرير بمحاذاة
  البكسل.
- **دعم جزئي للـ HTML** — الوسوم `<h1>`–`<h6>`, `<p>`, `<b>`,
  `<i>`, `<u>`, `<s>`, `<font color>`, `<a href>`, `<img alt>`,
  `<hr>`, `<br>`, `<pre>`, `<ul>`/`<ol>`/`<li>`, `<blockquote>`،
  إضافة إلى الجداول `<table>`/`<tr>`/`<td>`/`<th>` وبعض المُلخّصات
  (entities) مثل `&nbsp;` و`&amp;` و`&lt;` و`&gt;` و`&quot;`.
- **العربية** — قراءة ISO-8859-6، تشكيل الحروف حسب الربط
  (منفصلة/ابتدائية/وسطية/نهائية)، تراكيب "لا"، أرقام عربية-هندية.
- **الاتجاه الثنائي (BiDi L2)** — تُعاد ترتيب الكلمات العربية
  المتتالية داخل فقرة LTR بشكل صحيح، و`dir="rtl"` يجعل الفقرة
  RTL بالكامل.
- **المحاذاة** — `align="left|right|center"` على وسوم الفقرات
  والعناوين.
- **لوحة المفاتيح والفأرة** — التنقّل بـ Tab، والتفعيل بـ Enter/
  Space، والتمرير بالأسهم ومفاتيح الصفحة، ودعم الفأرة عبر PSG
  مباشرة.

## تنظيم الملفات

```
src/mwbrowser.asm        الشيفرة الرئيسية
src/iso8859_6.inc        جدول ISO-8859-6 ← رمز الخط + أعلام الربط
dist/mwbrowser.com       الملف التنفيذي المُجمَّع
```

## البناء

يتطلب [asMSX](https://github.com/gflorez/asmsx) و `mtools` لحقن الملفات
في القرص.

```sh
./tools/build.sh          # تجميع src/mwbrowser.asm إلى dist/mwbrowser.com
./tools/inject.sh         # كتابة dist/mwbrowser.com داخل قرص الإقلاع
./tools/run.sh            # تشغيل openMSX مع إدراج القرص
```

من موجّه MSX-DOS:

```
A> mwbrowsr
```

ثم اكتب اسم الملف في شريط العنوان (مثل `a:\test.htm`) واضغط Enter.

## اختصارات لوحة المفاتيح

| المفتاح           | الإجراء                       |
| ----------------- | ----------------------------- |
| Tab / Shift+Tab   | تدوير التركيز                 |
| Enter / Space     | تفعيل الزر أو الرابط          |
| الأسهم            | التمرير أو نقل التركيز        |
| Space أو M        | صفحة للأسفل                   |
| B                 | صفحة للأعلى                   |
| F1                | نافذة "حول"                    |
| Esc               | إغلاق الحوار / الخروج         |
